Introduction

Learn how to get started with our platform and build your first project in minutes.

Overview

Our platform provides a complete toolkit for building modern web applications. With built-in authentication, database management, and a powerful API, you can go from idea to production in record time.

💡 Tip: If you're new here, start with the Quick Start guide to build your first app in under 5 minutes.

Installation

Install the package using your preferred package manager:

npm install @your-package/core

Requirements

  • Node.js 18 or later
  • npm, yarn, or pnpm
  • A supported database (SQLite, PostgreSQL, or MySQL)

Basic Usage

Import and initialize the library in your application:

import { createApp } from '@your-package/core';

const app = createApp({
  database: 'sqlite',
  auth: { enabled: true },
});

app.listen(3000);

That's it! Your application is now running with authentication and a database connection ready to use.
